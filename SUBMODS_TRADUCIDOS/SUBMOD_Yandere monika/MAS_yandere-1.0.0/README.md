## 📝 Descripción
Este mod se centra en tener una Monika yandere, dándole al jugador una experiencia de una relación tóxica como se ve en animes de este estilo.

## ⭐ Este mod contiene 
- Nuevos diálogos
- Nuevas despedidas
- Nuevos cumplidos
